This is an APP (as single html file, download here) that search and show content of an ID (multiple formats supported, see below) only with P2P technologies.

Open the right menu for add content with the + button, and use share links.

Why it rocks

It's a single HTML page. Without any external references. It works if opened in localhost (file:///). Download here.
This is a project to promote the decentralization of contents in respect of net-neutrality and freedom. Feel free to share it or host it on your website.
Note: This page itself is shared in the same system, it act as home-page. Potentially, it can be a gateway to other HTML content in the same system.

Examples - Case study

Sintel.mp4

A video file. Viewed and shared in streaming.
A group of friends can use this system to share videos or films. It's possible to click Menu -> + , build an ID of a video file, share the link in a private WhatsApp group. Who view the content, help others as a seed.
Note: if opened on another PC in the same lan, is streamed over lan. This system can be used to share files between devices in lan, by sharing only ID/Url via IM like WhatsApp or Telegram.

torbrowser-install-win64-8.5.3_en-US.exe

GetTor robot provide alternative methods to download Tor Browser, specially for situations where access to Tor Project's official website and mirrors is blocked.

The link above (Windows bundle as example) search it in the p2p network. So, people can obtain Tor by know only the above ID and have access to a copy of search-id html page.

Can be better: it's possible to use a bitcoin address, and publish updates without need to change the known ID (that will remain fixed in time even if address an updated version). Read below in Types of ID supported section.

moonscape.html

A demo of an .html content that contain other resources loaded in the same way.
The html contain code like <code><iframe data-id='4d616494baf69a1ce2a8c00effac986a4c106da4'></iframe></code> and the system automatically convert the data-id attribute for nested contents.

This system can be used in combo with memo.cash, a Social Network based on Dapp build on Bitcoin Cash. Example: A post on memo.cash. The post ID is a BCH transaction, use ID here to render the content.

Types of ID supported

  • Magnet/torrent infohash

    Example: Video MP4, Sintel

  • Bitcoin (or Bitcoin Cash) address - Torrent hash is extracted from the latest outgoing transaction from the address ID, via OP_RETURN.

    This approach is based on elendirx/web2web idea.

    People can share a bitcoin address, that point to the content. After, people can update the content without need to change the ID, by adding a transaction.

    Example: This welcome.html you are reading. The ID is always the same, the content changes when we update it.

    If you want to publish something with this method, contact us in our Telegram group.

  • Bitcoin (or Bitcoin Cash) transaction

    Example: this (link to blockchain.com) Bitcoin transaction contain EW magnet:?xt=urn:btih:6a9759bffd5c0af65319979fb7832189f4f3c35d (via OP_RETURN), then the torrent is located via WebTorrent.

    This example was made with https://eternitywall.it/. Thanks.

    Example: Video MP4, Sintel

Types of content supported (inline viewer)

  • Video (with SRT and VTT subtitles)
  • Audio
  • PDF
  • Markdown (original + CommonMark + GitHub flavored) - sample/reference (Extension: .md , .md.txt)
  • BB-Code - sample/reference (Extension: .bbc , .bbcode, bbc.txt, bbcode.txt)
  • Any other kind of files as binary
We plan to support other kind of viewer, like gallery, wiki syntax, etc.

Parameters

  • autoplay - Recommended to false if used as embedded widget. Default: false
  • trackers - Trackers, for overwrite the defaults in bundle

Project files structure

  • search-id.min.html - Automatically builded by build.php. The unique, minified file that people can share.
  • build.php - Generate search-id.min.html based on search-id.html and resources - Launch to build a release.
  • search-id.html - Development version, HTML
  • embedding/* - Resources. build.php embed it in search-id.min.html
  • welcome.html - Project homepage, development version. It's this page you are reading. Release edition shared in the same system.
  • dev.html - Development tools, generally test links.

Domain search-id.org

The domain search-id.org run:

  • a copy of the above files
  • a WebTorrent and UDP/TCP tracker
  • a Pool server instance (experimental)
We also run a dedicated server as Pool client (experimental)
We expose all details https://search-id.org/stats and https://search-id.org/stats.json here in JSON.
Remember: anyone can host the single search-id.min.html file, publish a mirror, keep a copy in an usb key.
